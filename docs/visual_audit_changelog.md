# Visual audit changelog

В этой версии изменён только визуал и оформление. Видимый текст сайта не изменялся.

## Изменения

- Подставлены презентационные кропы для фоновых изображений и портретов.
- Заменён кроп кабины на чистый вариант без фрагментов текста слайда.
- Ослаблена фоновая сетка и уточнены градиенты первого экрана.
- Доработана шапка: больше воздуха, аккуратнее логотип и CTA.
- Усилена премиальность карточек, блоков, рамок и теней.
- Доработан блок “Экипаж”: пропорции портретов, карточки, контактная полоса.
- Улучшены адаптивные пропорции для планшетов и мобильных экранов.

## Не менялось

- Тексты.
- Структура страницы.
- Контакты.
- Ссылки CTA.
- Порядок секций.
