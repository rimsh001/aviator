# Исправления сайта АВИАТОР

Внесены правки по визуалу и заявке:

1. Выравнивание ширины сайта
   - добавлены ограничения ширины и защиты от горизонтального переполнения;
   - унифицированы контейнеры секций, сетки и контактный блок;
   - исправлены потенциальные переполнения в grid-блоках.

2. Фото Александры
   - для боковых карточек экипажа включён режим `object-fit: contain`;
   - увеличена высота зоны изображения;
   - фотография больше не должна обрезаться по лицу или корпусу.

3. Форма заявки
   - добавлена форма в финальный CTA-блок;
   - поля: имя, телефон, email/Telegram, управленческая задача;
   - так как сайт размещён на GitHub Pages без backend, форма открывает подготовленное письмо на `departure.aviator@gmail.com`.

Для полноценной отправки без почтового клиента позже можно подключить backend-обработчик, Telegram-бота, CRM, Google Sheets или сервис форм.
